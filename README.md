# AI for Envinronmental Data Science 
## Workshop 1: Intro to LLMs & Prompting Fundamentals

This repository contains materials for the Workshop on intro to large language models (LLMs) and prompting fundamentals for the [Masters in Environmental Data Science (MEDS)](https://bren.ucsb.edu/masters-programs/master-environmental-data-science/academics-meds-program), at the Bren School of Environmental Science & Management at UC Santa Barbara.

For more information visit the [MEDS AI for Environmental Data Science Workshop Series website](https://ucsb-meds.github.io/AI-for-EDS/).
